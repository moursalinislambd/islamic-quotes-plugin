== Description ==
The Islamic Quotes Plugin displays random Islamic quotes from a JSON file on your WordPress website. Users can share quotes via Facebook, Twitter, and WhatsApp. The plugin is compatible with both the Gutenberg editor and Elementor.

== Installation ==
1. Upload the `islamic-quotes-plugin` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the shortcode `[islamic_quote]` to display quotes in posts or pages.
4. The plugin also supports Gutenberg blocks and an Elementor widget.

== Frequently Asked Questions ==

= How do I add quotes? =
Edit the `quotes.json` file located in the `assets` folder to add or modify quotes.

= Can I customize the design? =
Yes, you can modify the CSS in the `style.css` file to match your theme's design.
